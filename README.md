The Daily Budget Tracking System is a web-based application designed to help users efficiently manage their daily income and expenses. Built using Python (Flask) and SQLite, the system allows users to record financial transactions, track spending habits, and monitor their overall balance in real time.

The application provides a simple and user-friendly interface where users can add income or expense entries along with descriptions. All transactions are securely stored in a SQLite database, and the system automatically calculates total income, total expenses, and the remaining balance.
